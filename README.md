# QQ Music API

> A Koa2 version of the QQ Music API that fetches QQ Music interface data through the web version.

## Installation

```
git clone git@github.com:Rain120/qq-music-api.git
cd qq-music-api
npm install
```

## Project Startup

```
npm run start

node app.js
```

The project listens on port `3200`.
